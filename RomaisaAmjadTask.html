Front-End WebDevelopment Bootcamp

Front-end web development bootcamp organized by MLSA FAST Peshawar is an intensive, fast-paced program designed to equip individuals with the practical skills needed to build websites and web applications. This bootcamp typically focus on hands-on learning, covering core technologies like HTML, CSS, JavaScript, and frameworks such as React JS with along with the highly experienced tea of MLSA.

Episode 1: Introduction to Web-Development and Git and GitHub

This Episode includes basic idea of web Deveopment; a short overview followed by Git and GitHub and it's set up and usage.

Episode 2: HTML

This episode is the level 1 for web Development.It includes basic Idea of HTML and use of basic tags.

Episode 3: CSS

This episodes includes the styling of a web page using CSS it's different properties and it's implementation on web page.

Episode 4: Bootstrap and Tailwind CSS

This episode includes the CSS frameworks which is use to make websites fastly.The speaker also shares more websites e.g. tailblocks , universe.io which can be use in making websites.
